### Run Locally

To run **portfolio** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/mymiracle0118/MyPorfolio.git
```

Windows:

```bash
git clone https://github.com/mymiracle0118/MyPorfolio.git
```
